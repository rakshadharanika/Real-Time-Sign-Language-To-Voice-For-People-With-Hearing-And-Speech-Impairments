# SignSpeak

Real-Time Sign Language to Voice and Text System  
for People with Hearing and Speech Impairments.

## Features
- Live camera-based sign language detection
- Sign → Text conversion
- Text → Voice output
- Multilingual support
- Accessible UI for deaf and mute users

## Tech Stack
- React + Vite (Frontend)
- MediaPipe & OpenCV (Sign Detection)
- TensorFlow (ML Model)
- Web Speech API (Voice)

## How to Run
```bash
npm install
npm run dev
